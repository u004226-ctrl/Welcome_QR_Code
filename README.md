# 📱 Welcome QR Code Generator

A simple, stylish, and open-source web application to generate and download QR codes instantly.

---

## 🇺🇸 English Version

### ✨ Features

- **Instant Generation:** Convert any text or URL into a high-quality QR code.
- **Easy Download:** Save your generated QR code as a PNG file.
- **Modern UI:** Features a dark theme, custom fonts, and a sleek modal interface.
- **Secret Easter Egg:** Double-tap the "X" (clear) button to visit X (Twitter).
- **Open Source:** Free to use and modify.

### 🛠️ Technologies Used

| Technology | Purpose |
|-----------|---------|
| **HTML5** | Structure |
| **CSS3** (Flexbox, Variables) | Styling & Layout |
| **JavaScript (ES6)** | Logic & Interactivity |
| **qrcode.js** | QR Code Generation |

### 🚀 How to Use

1. Enter your text or link in the input field.
2. Click **"Generate QR Code"**.
3. A modal will appear; click **"Download"** to save it or **"Close"** to exit.
4. Use the **"X"** button to quickly clear the input.

## 🇷🇺 Русская версия

### ✨ Возможности

- **Мгновенная генерация:** Превратите любой текст или ссылку в качественный QR-код.
- **Удобная загрузка:** Сохраняйте готовый QR-код в формате PNG одним нажатием.
- **Современный интерфейс:** Темная тема, кастомные шрифты и стильное модальное окно.
- **Секретная фишка:** Двойной клик по кнопке очистки (X) открывает соцсеть X (Twitter).
- **Open Source:** Свободное использование и изменение кода.

### 🛠️ Использованные технологии

| Технология | Назначение |
|-----------|---------|
| **HTML5** | Структура |
| **CSS3** (Flexbox, Переменные) | Стили и Разметка |
| **JavaScript (ES6)** | Логика и Интерактивность |
| **qrcode.js** | Генерация QR-кодов |

### 🚀 Как использовать

1. Введите текст или ссылку в поле ввода.
2. Нажмите кнопку **"Generate QR Code"**.
3. В появившемся окне нажмите **"Download"**, чтобы сохранить файл, или **"Close"**, чтобы закрыть окно.
4. Используйте кнопку **"X"** для быстрой очистки поля ввода.
